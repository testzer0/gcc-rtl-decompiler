Detailed Description

The base class from which all other classes inherit. Defines some (virtual) functions common to every class.

Prints the preorder traversal of the abstract syntax (sub)tree rooted at this node. Must not be overriden by inheriting classes: they must override PrintChildren instead. Prints only the name of the node (rest is printed in PrintChildren)
