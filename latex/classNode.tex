\doxysubsection{Detailed Description}
A class representing nodes in the abstract syntax tree. This class acts as a superclass for all token classes that are used in the A\+ST; i.\+e., each token class inherits from this class. All nodes have a pointer to their parent in the A\+ST, their lexical location, and a function to print the subtree rooted at them in-\/order, along with other members.
