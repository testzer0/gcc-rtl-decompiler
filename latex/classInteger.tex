\doxysubsection{Detailed Description}
A class representing integer variables in the program. The difference of \mbox{\hyperlink{classInteger}{Integer}} from T\+\_\+\+Int\+Constant is that \mbox{\hyperlink{classInteger}{Integer}} also allows negative integers. If the integer is negative then remember to pass -\/yylval.\+intconstant instead of positive.
